Información del Proyecto

Nuestro Equipo

José Antonio Montaño Gastélum es actualmente un estudiante de Ingeniería en Sistemas de Información en la Universidad de Sonora, con una fuerte pasión por el desarrollo web y las tecnologías emergentes. Desde temprana edad, José mostró un gran interés en la informática y la tecnología, lo que lo llevó a participar en varios talleres y eventos de tecnología durante su educación secundaria.

Durante su tiempo en la universidad, ha desarrollado habilidades en una variedad de lenguajes de programación incluyendo HTML, Python, y CSS. Además, ha trabajado en proyectos que utilizan frameworks y tecnologías modernas como Flask, Reflex, lo que le ha permitido construir aplicaciones web dinámicas y responsivas.

Tecnologías Utilizadas

Este proyecto ha sido desarrollado utilizando una variedad de tecnologías y herramientas modernas para asegurar eficiencia y funcionalidad óptima. La base del desarrollo se realizó con Python y Flask, un microframework poderoso y flexible para aplicaciones web. Además, se implementaron tecnologías front-end como HTML, CSS, y JavaScript para crear una interfaz de usuario interactiva y estéticamente agradable.

Para la recopilación de datos necesarios para el proyecto, se diseñaron dos web scrapers. El primero se encargaba de obtener una lista completa de revistas del catálogo objetivo, mientras que el segundo extraía datos específicos de cada revista. Ambos scrapers facilitaron la creación de dos archivos CSV, que almacenan la información recolectada de manera estructurada.

La estética del sitio se ha cuidado meticulosamente utilizando la fuente Montserrat 500 para garantizar legibilidad y un diseño cohesivo. La paleta de colores seleccionada incluye #1B3485 para la barra de navegación y el pie de página, #ecf0f1 para el texto de la barra de navegación, y #E1B640 como color de realce para los enlaces al pasar el cursor. Además, se utilizó FontAwesome para incorporar iconos que mejoran la interfaz visual del sitio.

Finalmente, se ha integrado el logotipo personal del desarrollador, aportando una marca distintiva y profesional al conjunto del proyecto. Este logotipo no solo refuerza la identidad visual del sitio, sino que también subraya la personalización y el enfoque detallado en el diseño del proyecto.
